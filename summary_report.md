## 說明

- **精確率 (Precision)** = `TP / (TP + FP)`
  - 在所有模型預測為「有物件」的結果中，有多少比例是真正有物件的。
- **召回率 (Recall)** = `TP / (TP + FN)`
  - 在所有真正存在的物件中，有多少比例被模型成功偵測出來。
- **IoU 閾值:** `0.5`
